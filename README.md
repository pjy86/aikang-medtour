# AiKang Medical Tour Website

A premium medical tourism website for AiKang (爱康医旅), specializing in providing healthcare services for international patients from the US and Indonesia seeking treatment in China.

## Features

- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **Multiple Pages**: Home, Services, About, Contact
- **WhatsApp Integration**: Direct contact via WhatsApp
- **Modern UI**: Clean, professional design with gradient effects
- **SEO Optimized**: Proper meta tags and semantic HTML

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deployment to Vercel

1. Push this code to a GitHub repository
2. Connect the repository to Vercel
3. Deploy automatically

Or use Vercel CLI:

```bash
npm i -g vercel
vercel
```
